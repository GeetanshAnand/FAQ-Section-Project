FAQ's

What is HTML and why is it used?

HTML (HyperText Markup Language) is the standard markup language used to create and design web pages. It structures content on the web using elements such as headings, paragraphs, links, images, and more.

What is the difference between a class and an ID in CSS?

A class can be used on multiple elements, while an ID should be unique and used on a single element. In CSS, classes are denoted by .classname and IDs by #idname.

What is the purpose of the DOCTYPE declaration in HTML?

The DOCTYPE declaration defines the version of HTML being used, ensuring the browser knows how to properly render the document.

How can I center a div using CSS?

You can center a div horizontally using margin: auto if the div has a set width. Another method is using Flexbox properties: display: flex; justify-content: center; align-items: center;.

What is the difference between == and === in JavaScript?

== compares values for equality after type conversion, while === checks for strict equality, meaning both value and type must be the same.

What is the purpose of JavaScript in web development?

JavaScript adds interactivity and dynamic behavior to web pages, such as responding to user inputs, manipulating the DOM, fetching data asynchronously, and more.

How can I create a responsive web page?

Use media queries in CSS, a flexible grid layout, and responsive units like percentages or em to adapt the design to different screen sizes.

What is the difference between let, const, and var in JavaScript?

var is function-scoped, while let and const are block-scoped. let allows value reassignment, while const does not allow reassignment once defined.

What is the DOM in JavaScript?

The DOM (Document Object Model) is a programming interface that represents the structure of an HTML or XML document as a tree structure, allowing scripts to update content, structure, and styles dynamically.

What is a CSS Flexbox and how is it used?

CSS Flexbox (Flexible Box Layout) is a layout model that allows items in a container to be aligned and distributed easily, providing flexibility for different screen sizes. It uses properties like display: flex, justify-content, align-items, and flex-direction to control the layout of elements.
